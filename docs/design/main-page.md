# 메인 페이지 ('/') 디자인 및 기획

## 디자인 컨셉
- 깔끔하고 현대적인 (Clean & Modern) UI
- 사용자 중심의 직관적인 정보 전달

## 구성 요소
1. **히어로 섹션 (Hero Section)**
   - 페이지 최상단 배치. 서비스 가치 제안.

2. **대학 리스트 섹션 (University List)**
   - 히어로 섹션 하단에 배치.
   - **데이터 소스**: Uyou 웹사이트에서 관리하는 대학 리스트(기본 정보).
   - **레이아웃**: 카드 그리드 형태.
   - **기능**: 각 카드 클릭 시 상세 페이지(`/detail/[university_id]`)로 라우팅.

## 상세 페이지 ('/detail/[university_id]') 기획
- **데이터 소스**: 해당 대학의 URL을 기반으로 크롤링된 게시글 리스트.
- **구성**: 
  - 상단: 대학 기본 정보
  - 하단: 크롤링된 입학/모집 게시글 리스트 (제목, 분류, 날짜 등)
  - 기능: 게시글 클릭 시 원문 URL로 이동.

## 이동 경로 (User Flow)
- 메인 페이지 ('/') 
  -> 대학 리스트 카드 클릭 
  -> 상세 페이지 ('/detail/[university_id]') 
  -> 해당 대학의 게시글 리스트 조회 및 원문 이동

## 디자인 가이드라인
- 모든 간격 및 크기는 4px 또는 6px 배수 적용.
